Hello World! now we get to the content of the webpage. This text will actually render on the screen when you open up a browser.

We can use tags like <div> within the body. A "div" specifies a division of the page, logically separating the content. There are two main types of tags - inline and block-level. divs are block level - they will start a new "block" on the next line. Remember, tags themselves don't get rendered but their content does.

When writing HTML markup and we want to use a reserved character like the less than sign (<), we have to escape it - otherwise the browser will interpret it as the start of a tag! Here's a list of some character escape sequences .

The <p> tag is another very common tag - it represents paragraphs.
The <span> tag is an inline tag - it will render text inline with the text around it.
If we want to make text bold, we use the <b> tag or the <strong> tag.
To italicize, use the <i> or <em> tag.

We can also nest tags within other tags, as we've been doing on this page so far. For example, head and body are always nested within the html tag. Indentation helps to make the nesting clear.

Now, how do we get to other webpages? Hyperlinks! We use the <a> or "anchor" tag to create hyperlinks to other resources / webpages. The anchor tag has an href attribute. An attribute is a property of a tag that consists of a key/value pair. It goes inside the tag itself. In the case of the above anchor "href" attribute, "href" is the key, and the URL "https://en.wikipedia.org/wiki/Hyperlink" is the value. The key and value are separated by an equals sign.

Another (optional) attribute of the anchor tag is "target", which can have the values: _blank, _parent, _self, or _top. If you specify _blank, the specified URL is open in a new window/tab.
