Welcome to Bootstrap

Styling all your pages with your own CSS seems like a lot of work, especially when you have to think about responsive design and all the different styles you'd have to write just to get something to look nice. Isn't there an easier way?

Yes! It's called Bootstrap! Bootstrap is a responsive, mobile-first CSS framework - it is a bunch of pre-written CSS styles (and optional JavaScript) that you can use. No more developing your own CSS from scratch - just use the styling that Bootstrap provides you by default, mainly by including classes on your HTML elements.

So, how to get started with Bootstrap? There's two main ways:

  1. Use their CDN (content delivery network) - just include a link tag with the right URL (that's what this page does)
  2. Download Bootstrap yourself and link to it locally

Note that using Bootstrap is not mutually exclusive with defining your own CSS - you can add your own stylesheet as well to customize, or even modify the Bootstrap source code since it is open-sourced.

Bootstrap is a large framework with many ready-to-go components and we won't go through all of them, but the most critical aspects of Bootstrap are:

  • The grid system - within a container, you can create rows, each of which can have 12 columns
  • Responsive by default - use different classes to collapse columns on smaller devices
  • Components - headings, lists, tables, forms, dropdown menus, modals and many others are nicely styled using Bootstrap classes
  • Utility classes - use various utility classes to style elements how you want them
  • Consistency - every element will look and feel unified in its design
